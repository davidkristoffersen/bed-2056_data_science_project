## Code

### Prerequisites

**Cli packages:**
- python3
- pip3

**Pip packages:**
- plotly
- pandas
- requests
- beautifulsoup4

**Install all pip packages:**  
```bash
$ pip3 install plotly pandas requests beautifulsoup4
```

### How to run

```bash
$ cd src
$ python3 main.py
```
