Overview and Motivation

In late 2019 and early 2020, the Coronavirus, originating from China, spread worldwide. Many nations enforced nationwide lockdowns in an attempt to combat the pandemic. Following these lockdowns, media across the world reported an increase in unemployment. This report will investigate these claims and figure out if there is a correlation between Coronavirus and unemployment. For simplicity's sake, we will focus on data from Norway and USA. We believe the majority of countries will have similar patterns.

We chose to focus on this topic because the pandemic is still on-going, and therefore very relevant. Additionally, we think it might show a variety of fascinating results.

Related Work

The interest in this topic came from the fact that Covid-19 is still on-going, and we were interested in doing our research regarding the effects of the virus on a nationwide scale. We chose unemployment because it is has been in focus by the media. Additionally, data regarding this topic should be abundant.

Initial Questions

This report will analyze if there is a possible correlation between the Coronavirus and unemployment rate, and if so, to what extent. Currently, as of writing this report, there have been two waves of Covid-19 in Noway and three in USA. The first wave, for both countries, occurred in late February/early March 2020. USA's second outbreak occurred in the summer, and Norway's second and USA's third outbreaks started in October and are still on-going.

Data

Since we have a variety of categories, we have used several sources that we will present below.

Unemployment rate Norway

The data for the unemployment rate in Norway is retrieved from Nav. Nav offers many great data sets, which can be found here: nav.no .

The specific data set we used for unemployment was: Nav unemployment .

The data set comes in excel format. To scrape it, we used the python library pandas: pandas.pydata.org , which supports reading excel formatted files.

Unemployment rate USA

The unemployment rate data for USA comes from the U.S Bureau of Labor Statistics: bls.gov . To retrieve the data, we do a post request to their API: api.bls.gov , where we pass some parameters which specify what data we want. The result comes in JSON format, and we use the json module for python to parse the data and extract the relevant information.

Covid cases

The data for the number of daily Covid cases is fetched from the European Centre for Disease Prevention and Control: ecdc.europa.eu . They supply a variety of data sets involving Covid, amongst other things. The data we use can be found here: ecdc Covid . The data is in CSV format, and we can parse the data by first separating the lines, before separating the rows by commas to extract the values for Norway and USA.

In each plot, there is also a toolbar with a variety of tools.

In USA, three significant waves of Covid are identifiable, where the last wave has the highest amount of cases and is still on-going (as of writing this). (Figure 4. Unemployment rate for Norway and USA)

Following the unemployment rate from 2015, we can see that it is slowly decreasing until the lockdown events in March 2020.

(Figure 5. Difference in unemployment rate between Norway and USA)

This plot shows that the difference in unemployment between USA and Norway was steadily decreasing in the years prior to Covid-19. Since the lockdown in Norway was enforced earlier than USA we see the difference go into the negative, meaning Norway had a higher unemployment rate than USA. However, as soon as USA enforced their lockdown the graph went back up to USA having a significantly higher unemployment rate. Since the inital lockdowns the difference has been decreasing, however it is still significantly higher than what it was prior to Covid-19.

Final Analysis

Based on the data, there appears to be a correlation between the number of Covid cases and unemployment rate. Looking at the plot for confirmed cases of Covid-19, we can see that when the covid cases increased and lockdowns were enforced, the unemployment also increased. Additionally, comparing the relative Covid cases graph (figure 4) and the relative unemployment graph (figure 5), we can see that USA had more confirmed cases, as well as higher unemployment rate, compared to Norway. This leads us to believe that there is a correlation between the number of Covid cases and the unemployment rate.

Interestingly, in the following waves of Covid, no noticeable rise in unemployment was detected. We suspect this is the case for two main reasons:
  • People who lost their jobs found new jobs that did not require lay off in the case of an increase in Covid cases.
  • No lockdowns have been enforced in the second (and third in the case of USA) wave.
At the time of writing this, no second lockdown has been enforced. If a second lockdown were to be enforced, we would expect to see an increase in unemployment again, however not as severe as the first.
