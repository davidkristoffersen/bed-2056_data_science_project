Overview and Motivation

In late 2019 and early 2020 the Corona virus, originating from China, spread across the world. This lead to a global pandemic, where, in order to combat the spread of the virus, many nations enforced nation wide lockdowns. Following these lockdowns, media across the world reported an increase in unemployment. This report will investigate these claims, and try to figure out if there is a correlation between the Corona virus and unemployment. For simplicity's sake, we will focus on data from Norway and USA. We believe the majority of countries will have similar patterns.

We chose to focus on this topic because the pandemic is still on-going, and therefore very relevant. In addition, we think it might show a variety of interesting results.

Related Work

The interest for this topic came from the fact that Covid-19 is still on-going, and we were interested in doing our own research regarding the effects of the virus on a nation wide scale. We chose unemployment because it is has been in focus by the media. Additionally, data regarding this topic should be abundant.

Initial Questions

The questions this report will try to answer are: is there a real correlation between the Corona virus and unemployment rate, and if so, to what extent. Currently, as of writing this report there have been two waves of Covid-19 in Noway, and three in USA. The first wave, for both countries, occured in late February/early March 2020. America's second outbreak occured in the summer, and Norway's second and America's third started in October, and is still going on.

Data

Since we have a variety of categories, we have used several sources that we will present below.

Unemployment rate Norway

The data for unemployment rate in Norway is retrieved from Nav. Nav offers many great data sets, which can be found here: nav.no

The specific data set we used for unemployment was: Nav unemployment

The data set comes in excel format, to scrape it we used the python library pandas: pandas.pydata.org , which support reading excel formatted files.

Unemployment rate USA

The unempoyment rate data for USA comes from the U.S Bureau of Labor Statistics: bls.gov . To retrieve the data, we do a post request to their api: api.bls.gov , where we pass some parameters which specify what data we want. The result comes in JSON format, and we use the json python module to iterate and parse the data.

Covid cases

The data for the number of daily Covid cases is fetched from the European Centre for Disease Prevention and Control: ecdc.europa.eu . They supply a variety of data sets involving Covid, amongst other things. The data we use can be found here: ecdc Covid . The data is in CSV format, and we can parse the data by separating the lines, and then separating the rows by comma to extract the values we are interested in.

Final Analysis

Based on the data, there appears to be a correlation between the number of Covid cases and unemployment rate. From the figures, we can see that as soon as the number of Covid cases increased, and lockdowns were enforced, the unemployment rate also increased. For the Norwegian unemployment data (figure 1), the plot might make it appear as though the unemployment rate increased before the lockdown, however this is because both the lockdown and the unemployment-peak were in the same month. In actuality, the unemployment rate followed the lockdown. Interestingly, in the following waves of Covid, no noticeable difference in unemployment was detected. We suspect this is the case for two main reasons:
  • People who lost their jobs found new jobs that did not require lay off in the case of an increase in Covid cases.
  • No lockdowns have been enforced in the second (and third in the case of USA) wave.
At the time of writing this, a second lockdown has not been enforced. However, if a second lockdown were to be enforced, we would expect too see an increase in unemployment again, although not as severe as the first.
